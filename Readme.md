Check if primitive.

## Installation

    $ npm install @f/is-primitive

## Usage

```js
var isPrimitive = require('@f/is-primitive')

isPrimitive(1) // => true
```

## API

### isPrimitive(val)

- `val` - value to check

**Returns:** boolean indicating if `val` is a primitive
